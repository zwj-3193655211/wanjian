# 万剑归宗 - 手势识别游戏项目计划

## 项目概述
这是一个基于 YOLO11 手势识别的"万剑归宗"互动游戏。玩家通过手势控制屏幕上飞剑的发射和方向，体验武侠小说中"万剑归宗"的震撼效果。

---

## 技术栈
- **Python 3.13.7** - 主开发语言
- **Ultralytics YOLO11** - 最新版手势识别（手部21个关键点检测）
- **OpenCV** - 摄像头捕获和图像处理
- **Pygame** - 游戏窗口和粒子效果渲染

## 手势控制设计

| 手势 | 动作 |
|------|------|
| 张开手掌 | 激活"万剑归宗"模式 - 多把飞剑从屏幕四周汇聚到手掌位置 |
| 握拳 | 蓄力效果 - 飞剑能量聚集 |
| 移动手掌 | 控制飞剑目标位置 |
| 两指捏合 | 单发飞剑 |

## 关键技术点

### YOLO11 手部关键点
YOLO11 可检测手部 21 个关键点：
- 0: 手腕
- 1-4: 大拇指
- 5-8: 食指
- 9-12: 中指
- 13-16: 无名指
- 17-20: 小指

### 飞剑粒子效果
- 使用 Pygame 粒子系统
- 每把飞剑带有拖尾效果
- 飞行轨迹可设置曲率（剑气效果）
